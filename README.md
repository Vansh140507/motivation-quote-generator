# 💡 Motivation Quote Generator

A simple and interactive web application that displays random motivational quotes with a single click. Built using **HTML, CSS, and JavaScript**, this project focuses on DOM manipulation and basic UI design.

---

## 🚀 Features

* 🎯 Displays random motivational quotes
* 🔄 Button click to generate new quotes
* 🎨 Clean and responsive UI design
* ⚡ Lightweight and fast (no frameworks)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* Quotes are stored in a JavaScript array
* A random index is generated using `Math.random()`
* The quote text updates dynamically using DOM manipulation

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Vansh140507/motivation-quote-generator.git
```

2. Open `index.html` in your browser
